# Nano (MCP mcp-kv.ru) — обложка и баннер под «Вордпресс статьи»

Соответствие blueprint **RU SEO/GEO 2026**: обложка **16:9**, горизонтальный баннер в тексте **21:9**, разрешение по умолчанию **2K**, формат **png** для `nano_banana_pro` (или по политике заказчика).

## Обложка 16:9 (featured image)

Базовый смысл промпта (передаётся в `nano_banana_*` как `prompt`):

- Фотореалистичная или чистая editorial-иллюстрация под тему статьи для **блога WordPress / SEO / технической веб-разработки**.
- Композиция широкая **16:9**, читаемая миниатюра в ленте записей.
- Короткий заголовок на **кириллице** (сокращённая формулировка темы, не весь SEO-title).
- Без логотипов сторонних мессенджеров, без водяных знаков чужих школ, без вымышленных сертификатов.
- Если есть **`image_input`** с лицом автора — **не искажать лицо** (`identity_lock`); можно менять одежду/фон/жест.

Переопределение через переменную окружения **`NANO_COVER_PROMPT`** (полная замена текста промпта).

## Баннер 21:9 (в теле статьи)

- Широкий горизонтальный баннер **21:9**, единая визуальная тема с материалом.
- Стиль: спокойный UI/абстракт / рабочее место / интерфейс редактора — по теме статьи; без посторонних URL на картинке.
- Кириллица допускается только если это короткий нейтральный заголовок темы.

Переопределение: **`NANO_BANNER_PROMPT`**.

## После генерации

1. Забрать временный `https://…` из ответа инструмента.
2. Вызвать **`wordpress_upload_media`** (`file_url`) → использовать **постоянный** URL с домена сайта в HTML.
3. Для обложки сохранить **`featured_media`** и связать с постом.
